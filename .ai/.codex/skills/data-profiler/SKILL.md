# Data Profiler

## Workflow

- Load the dataset and confirm the profiling tool (prefer ydata-profiling if available).
- Generate a full profile report and basic data catalog metadata.
- Save the profile to `reports/profiles/` with a dated filename.
- Summarize key distributions, missingness, and anomalies.

## Output

- Profile report and a short catalog entry summary.
