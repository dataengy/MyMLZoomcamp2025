# Test Coverage

## Workflow

- Run pytest with coverage for the relevant package or module.
- Generate HTML and summary reports under `reports/coverage/`.
- Identify low-coverage files and propose targeted tests.
- Note any coverage exclusions or flaky areas.

## Output

- Coverage report location and prioritized test suggestions.
