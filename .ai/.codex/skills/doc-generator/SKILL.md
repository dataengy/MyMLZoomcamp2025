# Documentation Generator

## Workflow

- Identify the target docs (README, docs pages, API reference).
- Extract docstrings or metadata from `src/`.
- Generate or update docs with clear examples.
- Keep changes scoped and avoid duplicating existing content.

## Output

- Updated documentation and a brief change note.
