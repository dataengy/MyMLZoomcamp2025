# Report Builder

## Workflow

- Collect metrics from `reports/experiments/`, `logs/`, or user-provided sources.
- Build concise tables and charts for key metrics.
- Produce an HTML or PDF report under `reports/` with a dated filename.
- Summarize results and recommended next actions.

## Output

- Report artifact and a short executive summary.
