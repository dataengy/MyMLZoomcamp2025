# Integration Tests

## Workflow

- Identify system boundaries and required external services.
- Create fixtures and mocks for external dependencies.
- Implement tests under `tests/integration/`.
- Provide a lightweight test database or data stubs if needed.
- Document how to run the tests.

## Output

- New or updated integration tests and a run command.
